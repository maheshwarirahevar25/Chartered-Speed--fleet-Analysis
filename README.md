# 🚌 Chartered Speed — Fleet & Driver Performance Analysis

## 📌 Project Overview
A complete data analysis project for Chartered Speed (passenger mobility company) 
covering fleet monitoring, driver violation tracking, and fuel consumption analysis 
using SQL, Excel, and Power BI.

## 🎯 Objective
To analyze driver compliance data (overspeeding, harsh braking, mobile usage, 
seatbelt violations) and bus performance metrics (fuel consumption, average speed) 
to help management make data-driven decisions.

## 🛠️ Tools Used
- **SQL (MySQL/DBeaver)** — Data querying and analysis
- **Excel** — Data cleaning, Pivot Tables, Charts
- **Power BI** — Interactive dashboard and reporting

## 📊 Dataset
Fleet data of 30+ buses across Ahmedabad, Jaipur, Bikaner, and Surat with columns:
Driver_Name, Bus_ID, City, Route, Violation_Type, Speed_kmh, Fuel_Consumed_L, 
Date, Month, Shift

## 🔍 Key Insights
- Identified top 5 drivers with highest fuel consumption
- Found city-wise violation patterns (Jaipur had highest violations)
- Tracked monthly violation trends (Jan–Mar 2026)
- Compared Morning vs Evening shift performance

## 📈 Power BI Dashboard Features
- 4 KPI Cards: Total Buses, Total Fuel, Total Violations, Avg Speed
- Line Chart: Driver count by month
- Donut Chart: Violation by type
- Bar Chart: Violation by city
- Stacked Bar Chart: Fuel consumed by route
- Slicers: City and Date Range filters

## 📋 Excel Analysis
- Pivot Tables for City, Route, and Violation Type summaries
- Conditional Formatting to highlight overspeeding drivers
- Charts: Pie, Column, and Bar charts

## 💻 SQL Queries
Sample queries used for analysis:
- Top drivers by overspeeding count
- City-wise violation count
- Monthly violation trends
- Average speed by city
